🎙️ Meeting Insight Dashboard (Full-Stack AI)


A modern, full-stack transcription platform that leverages OpenAI's Whisper model to transform meeting audio into searchable, organized text. This project was built to provide a seamless "Local-to-UI" pipeline for high-accuracy speech-to-text processing.

🚀 Key Features

AI-Powered Transcription: Uses the Whisper base model for high-speed, accurate speech recognition.

Dynamic UI: A React-based dashboard featuring a "Clean Upload" logic that resets the view for every new session.

Transcription History: A dedicated sidebar to track and review previous transcripts within the session.

Automated Audio Pipeline: Backend automatically handles file uploads and uses FFmpeg to convert various formats into the required 16kHz WAV standard.

Accessibility First: Integrated accessibility toolbar for high contrast and dyslexia-friendly viewing.

🛠️ Tech Stack
Frontend: React, Vite, Tailwind CSS, Lucide Icons (via Lovable).

Backend: Node.js, Express.js.

Processing: FFmpeg (Format conversion), OpenAI Whisper (AI Inference).

Version Control: Git & GitHub.
